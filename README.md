# HackQuest AI

HackQuest AI is a futuristic AI + Web3 hackathon platform where developers complete quests, gain XP, and climb live leaderboards.

## Features

- Responsive dashboard and page layouts
- Quest system with XP progression
- Live-style activity feed and leaderboard UI
- Animated premium interface built with React, Tailwind, and Motion

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Motion (Framer Motion)

## Getting Started

Prerequisites:

- Node.js 18+

Install and run:

1. npm install
2. Copy .env.example to .env.local and set GEMINI_API_KEY if required
3. npm run dev

Build for production:

1. npm run build
2. npm run preview
